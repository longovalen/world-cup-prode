📋 Migration Steps:

  1. From index-flag.html: Open https://longovalen.github.io/world-cup-prode/index-flag.html
  2. Open this migration tool: https://longovalen.github.io/world-cup-prode/migrate-data.html
  3. Click "Export Data (Backup)" to download the data
  4. Rename file in GitHub: Rename index-flag.html to index.html
  5. From new index.html: Open https://longovalen.github.io/world-cup-prode/
  6. Open this migration tool again
  7. Click "Import Data (Restore)" and select the downloaded file
  8. Verify data was imported correctly
